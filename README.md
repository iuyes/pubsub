 项目首页

###一个publish/subscribe的js独立库

`无需依赖第3方`

##API
 
1. ps.pub     /  ps.publish     发布事件通知
2. ps.sub    /   ps.subscribe   订阅通知
3. ps.unsub  /   ps.unsubscribe 取消已订阅的事件通知
